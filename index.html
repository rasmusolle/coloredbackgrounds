Colored Desktop Backgrounds

This is a site which contains a collection of colored desktop backgrounds. Simply save the color you want as an image and set it as your desktop background.

Most of these are taken from the standard HTML color names.

But my OS has colored backgrounds! Why does this exist?

Congratulations! Not every OS has colored backgrounds, especially not very many. I made this especially because Chrome OS didn't have a big enough selection of colored backgrounds.
